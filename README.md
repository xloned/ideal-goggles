# Лабораторная работа №1. Принципы создания подсистемы ручного ввода информации в информационных системах

## Стек технологий

- **Backend:** Python 3, Django 6
- **Frontend:** HTML, CSS, JavaScript
- **База данных:** SQLite3
- **Язык интерфейса:** Русский

## Задание

Разработать согласно требованиям и реализовать интерфейс автоматизированного рабочего места оператора по продажам для выполнения следующих действий:

1. Открыть базу клиентов
2. Ввод имени клиента, при вводе имени осуществляется поиск по списку клиентов, и если такой находится, то вывести:
   - общий счет покупок клиента (сумма всех покупок)
   - текущий счет клиента (деньги на счету фирмы для покупок)
   - потолок кредита (кредитный лимит, который не изменяется)
   - текущий долг клиента
   - остаток кредита (разница между потолком кредита и текущим долгом)
   - комментарий (о причине долга клиента, его надежности и т.п.)
3. Открыть базу товаров
4. Создать новый заказ с таблицей склада товаров
5. Выбор вида продажи:
   - **Наличный расчет** — увеличивается общий счет покупок клиента, количество товаров уменьшается
   - **Безналичный расчет** — увеличивается общий счет покупок, уменьшается текущий счет клиента, количество товаров уменьшается
   - **Кредит** — увеличивается общий счет покупок и текущий долг, уменьшается остаток кредита; при приближении долга к 90% потолка кредита подсвечивается красным
   - **Бартер** — обмен товаров без денег, стоимость отданных = стоимости полученных
   - **Взаимозачет** — уменьшается текущий долг, увеличивается остаток кредита, товары возвращаются на склад
6. Ввод перечня товаров и количества в табличной части с автоматическим указанием цены, суммы и текущего остатка
7. Цветовая индикация остатка: зеленый если хватает, красный если превышает + звуковое предупреждение
8. Автоматический подсчет общей суммы и построение отчета по клиентам
9. Предупреждения при превышении кредитного лимита и при незаполненных строках

## Запуск

```bash
pip install django
python manage.py migrate
python manage.py runserver
```

Приложение будет доступно по адресу: http://127.0.0.1:8000/
